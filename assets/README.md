# Assets 资源文件

项目相关的静态资源文件存放目录

## 📁 目录结构

```
assets/
├── images/          # 项目图片资源
├── diagrams/        # 架构图、流程图等
└── presentations/   # 演示文稿、PPT等
```

## 📝 使用规范

- **图片格式**：优先使用 PNG/SVG，支持 JPG
- **文件命名**：使用小写字母和连字符，如 `cose-architecture.png`
- **文件大小**：单个文件不超过 5MB
- **引用方式**：相对路径引用，如 `![架构图](assets/diagrams/architecture.svg)`

## 🎯 内容分类

### 📸 Images
- Logo 和品牌素材
- 产品截图
- 团队照片
- 其他项目相关图片

### 📊 Diagrams  
- 技术架构图
- 商业模式图
- 流程图表
- 数据可视化

### 📽️ Presentations
- 路演 PPT
- 技术分享
- 商业计划演示
- 会议资料
